# TP1-Front

El proyecto es un generador de contraseñas, el cuál cuenta con una base de datos para poder almacenar las contraseñas generadas en base al usuario. Además, la contraseña será generada en base a ciertos parámetros elegidos por el usuario (cantidad de caracteres, mayusculas, minusculas, simbolos y números), y mostrará que tan segura es.

Para ejecutar este proyecto, abre este link:
https://tp-1-front.vercel.app/
O corre en la consola (se abrirá la página automáticamente):
cd password-generetor-pro
npm install
npm run dev
